Para implementar o Delta Encoding aliado a Codificação de Huffman em fontes de informação, basta ter os ficheiros que se pretendem comprimir no mesmo diretório do Delta+Huffman.py, bem como ter o ficheiro huffmancodec.py nesse mesmo diretório. Tendo estas condições reunidas, para comprimir um ficheiro de nome "x" é apenas necessário alterar na main do Delta+Huffman.py na parte que diz "file = ..." para "file = x".

No final, o ficheiro que esta combinação irá criar terá um nome do tipo "xDELTA+HUFFMAN.dat"
