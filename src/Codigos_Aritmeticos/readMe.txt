Para implementar os Códigos Aritméticos em fontes de informação, basta ter os ficheiros que se pretendem comprimir no mesmo diretório do Codigos_Aritmeticos.py, bem como ter o ficheiro arithmeticcoding.py nesse mesmo diretório. Tendo estas condições reunidas, para comprimir um ficheiro de nome "x" é apenas necessário alterar na main launcher do Codigos_Aritmeticos.py na parte que diz "main(..., ...)" para "main(x, nome do ficheiro de saída)".
