Para implementar o algoritmo LZW em fontes de informação, basta ter os ficheiros que se pretendem comprimir no mesmo diretório da pasta .idea do LZW. Tendo estas condições reunidas, para comprimir um ficheiro de nome "x" é apenas necessário alterar na main do MyComp.java na parte que diz "filename = ..."; para "filename = x".

No final, o ficheiro que esta combinação irá criar terá um nome do tipo "x.lzw"
