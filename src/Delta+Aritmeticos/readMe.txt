Para implementar o Delta Encoding aliado a Códigos Aritméticos em fontes de informação, basta ter os ficheiros que se pretendem comprimir no mesmo diretório do Delta+Aritmeticos.py, bem como ter o ficheiro arithmeticcoding.py nesse mesmo diretório. Tendo estas condições reunidas, para comprimir um ficheiro de nome "x" é apenas necessário alterar na main do Delta+Aritmeticos.py na parte que diz "nome = ..." para "nome = x".

No final, o ficheiro que esta combinação irá criar terá um nome do tipo "xDELTA+ARITMETICO.dat"
